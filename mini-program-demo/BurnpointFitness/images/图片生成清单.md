## 使用方式

1. 生成图片后放入 `burnpoint-fitness/images/` 目录
2. 在 HTML 中找到对应元素的 CSS class，将 `background: linear-gradient(...)` 替换为 `background-image: url('images/xxx.jpg')`
3. 头像：将 `.cm-avatar-X` 和 `.profile-avatar` 的渐变背景改为 `background-image`

**当前 Demo 纯 CSS 已经足够展示功能。** 图片只是锦上添花，不是必须。
